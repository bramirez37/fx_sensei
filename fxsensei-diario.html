Guia de Utilização
Como usar o diário — método Juliano v4
Versão Cloud

Os teus dados estão guardados na cloud (Supabase). Podes aceder a partir de qualquer dispositivo com a tua conta. Os dados ficam associados ao teu email — nunca se perdem se limpares o browser.

"Se não consegues explicar este trade a outra pessoa em 30 segundos com clareza, ainda não estás pronto para o executar." — Juliano

Rotina recomendada
  1. Abre primeiro o separador Pre-Trade e preenche o contexto completo.
  2. Confirma score, warnings e decisão antes de qualquer entrada.
  3. Depois de fechares a operação, regista o Pós-Trade e liga-o ao pré-trade correspondente.
  4. Consulta o History para rever análises e o Dashboard para padrões e estatísticas.
Debug rápido
  • Se ficar em loading, abre a consola do browser e procura erros vermelhos.
  • Valida login, leitura de sessão e acesso às tabelas pretrades e trades.
  • Confirma que o schema Supabase foi executado sem erros e com RLS ativo.
  • Usa o botão Debug no login para correr diagnóstico básico local.
